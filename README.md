# Pava Photo Studio

Сайт фотостудії без фотографа в Дніпрі. Dark-theme, адаптивний, з онлайн-бронюванням.

---

## Стек

| Шар | Технологія |
|---|---|
| Сервер | Node.js + Express 5 |
| База даних | SQLite 3 (`data/bookings.db`) |
| Фронтенд | Vanilla HTML / CSS / JS (без фреймворків) |
| Шрифти | Google Fonts  Playfair Display, Inter |

---

## Функціонал

### UI / UX
- Dark-theme з фіолетовим акцентом (`#8c3272`)
- Тонкий радіальний градієнт на тлі
- Sticky-навбар з **бургер-меню** на мобільних ( 900 px)
- Анімації появи секцій при скролі (IntersectionObserver)
- Parallax-ефект на фоні hero
- Typed-ефект у підзаголовку hero
- Hover-ефекти на FAQ-елементах
- Повністю адаптивна верстка (мобільний / планшет / десктоп)

### Секції
- **Hero**  логотип, заголовок, кнопки CTA
- **Про студію**  зони, характеристики
- **Ціни**  таблиця тарифів зі знижкою
- **Галерея**  масонрі-сітка з lazy-load та кнопкою Завантажити ще
- **Відгуки**  карусель з автоскролом
- **FAQ**  акордеон з hover-ефектом
- **Правила**  модальне вікно
- **Бронювання**  модальна форма з вибором дати / часу
- **Футер**  контакти, соцмережі, карта

### Бекенд API

| Метод | Шлях | Опис |
|---|---|---|
| `GET` | `/api/bookings?limit=N` | Список бронювань (сортування за датою) |
| `POST` | `/api/bookings` | Нове бронювання |

**POST body (JSON):**
```json
{
  "name": "Ім'я клієнта",
  "phone": "+380...",
  "date": "2026-03-01",
  "time": "14:00",
  "notes": "коментар (необов'язково)"
}
```

Захист від подвійного бронювання  унікальна пара `(date, time)` в БД.  
При конфлікті повертає `409 Conflict`.

---

## Запуск

```bash
npm install
npm start
```

Сервер підніметься на `http://localhost:3000`.  
Порт можна змінити через змінну середовища: `PORT=8080 npm start`

## Структура проєкту

```
 server.js               # Express-сервер + SQLite API
 data/
    bookings.db         # SQLite (створюється автоматично)
 public/
     index.html          # Головна сторінка
     booking.html        # Модаль бронювання
     rules.html          # Модаль правил
     styles.css          # Всі стилі
     app.js              # Логіка бронювання, модалей, typed-ефект
     gallery.js          # Галерея: lazy-load, масонрі, ще
     interactions.js     # Бургер-меню, scroll-reveal, parallax, tilt
     images/
         logo.png
         logo-white.png
         main/           # Фото для hero
         gallery/        # Фото галереї
         background/     # Фото бічних панелей
```
