# Pava Photostudio

MVP-сайт фотостудії без фотографа з чорно-білим стилем, галереєю та записами клієнтів.

## Що реалізовано

- чорно-білий UI (більшість елементів у монохромній палітрі);
- галерея студії з 6 прев’ю-зонами;
- форма запису клієнта;
- API для записів (`GET /api/bookings`, `POST /api/bookings`);
- збереження записів у SQLite (`data/bookings.db`);
- захист від подвійного бронювання одного слоту (унікальна пара дата+час).

## Запуск

```bash
npm install
npm start
```

Після запуску відкрийте:

- http://localhost:3000

## Структура

- `server.js` — Express + SQLite API;
- `public/index.html` — сторінка сайту;
- `public/styles.css` — стилі;
- `public/app.js` — фронтенд-логіка для роботи з API.
